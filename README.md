📊 Customer Data Visualization Project

🎯 Project Objective
This project analyzes customer data using Python.
Various visualizations are created to understand customer behavior and spending patterns.

🛠 Tools Used
- Python
- Pandas
- Matplotlib
- Seaborn

📈 Visualizations Created
1. Age Distribution (Histogram)
2. Gender Distribution (Pie Chart)
3. Customers by City (Bar Chart)
4. Monthly Income Distribution (Box Plot)
5. Spending Score Analysis
6. Purchase Trends (Line Chart)
7. Product Category Analysis
8. Income vs Spending Score (Scatter Plot)
9. Daily and Monthly Sales Trends
10. Gender and Category Comparisons

📂 Dataset
Customer dataset containing:
- Age
- Gender
- City
- Monthly Income
- Spending Score
- Purchase Amount
- Product Category
- Purchase Date

🚀 Conclusion
The project helps understand customer demographics, income distribution,
spending behavior, and sales trends using visual analysis.
